1. Limits

The limit of f(x) as x approaches a is L, written limx→a f(x) = L, if f(x) can be made arbitrarily close to L by taking x sufficiently close to a (but not equal to a).

1.2 L'Hopital's Rule

If limx→a f(x)/g(x) gives the indeterminate form 0/0 or ∞/∞, then:

limx→a f(x)/g(x) = limx→a f'(x)/g'(x) (provided the right-hand limit exists)
Example: limx→0 (ex - 1 - x) / x2
This is 0/0. Apply L'Hopital: limx→0 (ex - 1) / 2x — still 0/0.
Apply again: limx→0 ex / 2 = 1/2

1.3 Squeeze Theorem

If g(x) ≤ f(x) ≤ h(x) near a, and limx→a g(x) = limx→a h(x) = L, then limx→a f(x) = L.

2. Continuity and Differentiability

2.1 Continuity

f(x) is continuous at x = a if: (1) f(a) is defined, (2) limx→a f(x) exists, (3) limx→a f(x) = f(a).

2.2 Differentiability

f'(a) = limh→0 [f(a + h) - f(a)] / h
StatementTrue/False
Differentiable implies continuousTrue (always)
Continuous implies differentiableFalse (e.g., f(x) = |x| at x = 0)
Discontinuous implies not differentiableTrue (contrapositive of above)

3. Mean Value Theorems

3.1 Rolle's Theorem

If f is continuous on [a, b], differentiable on (a, b), and f(a) = f(b), then there exists c ∈ (a, b) such that f'(c) = 0.

3.2 Lagrange's Mean Value Theorem (MVT)

If f is continuous on [a, b] and differentiable on (a, b), then there exists c in (a, b) such that:
f'(c) = [f(b) - f(a)] / (b - a)

Rolle's theorem is a special case of MVT where f(a) = f(b).

3.3 Cauchy's Mean Value Theorem

If f and g are continuous on [a, b], differentiable on (a, b), and g'(x) ≠ 0 on (a, b), then:
f'(c) / g'(c) = [f(b) - f(a)] / [g(b) - g(a)] for some c in (a, b)

Lagrange's MVT is a special case with g(x) = x.

Example: Verify MVT for f(x) = x2 on [1, 3].
f(1) = 1, f(3) = 9. [f(3) - f(1)]/(3 - 1) = 8/2 = 4.
f'(x) = 2x. Setting f'(c) = 4: 2c = 4 ⇒ c = 2.
Since c = 2 ∈ (1, 3), MVT is verified.

4.2 First Derivative Test

Sign of f'(x) before cSign of f'(x) after cConclusion
+-Local maximum
-+Local minimum
++Neither (increasing through c)
--Neither (decreasing through c)
Example: Find extrema of f(x) = x3 - 3x + 2.
f'(x) = 3x2 - 3 = 3(x-1)(x+1) = 0 ⇒ x = ±1
f''(x) = 6x
f''(1) = 6 > 0 ⇒ local minimum at x = 1, f(1) = 0
f''(-1) = -6 < 0 ⇒ local maximum at x = -1, f(-1) = 4

5.2 Techniques of Integration

Substitution (u-substitution)

∫ f(g(x)) g'(x) dx = ∫ f(u) du where u = g(x)
Example: ∫ 2x · ex² dx
Let u = x2, du = 2x dx
= ∫ eu du = eu + C = ex² + C

Integration by Parts

∫ u dv = uv - ∫ v du

LIATE rule for choosing u (in priority order): Logarithmic, Inverse trig, Algebraic, Trigonometric, Exponential.

Example: ∫ x ex dx
Let u = x, dv = ex dx ⇒ du = dx, v = ex
= xex - ∫ ex dx = xex - ex + C = ex(x - 1) + C

7. Key GATE Takeaways

  • Memorize standard limits, especially sin(x)/x and (ex - 1)/x.
  • L'Hopital's rule: only for 0/0 or ∞/∞ forms. Check the form before applying.
  • Differentiable ⇒ Continuous, but not the converse. Know counterexamples.
  • MVT and Rolle's theorem: check hypotheses (continuity and differentiability).
  • For maxima/minima: second derivative test is fastest; use first derivative test when f'' = 0.
  • Integration by parts with LIATE ordering. Know reduction formulas.
  • Taylor series are useful for evaluating limits and approximations.
  • For definite integrals, exploit symmetry (even/odd functions) to simplify.

Combinational Circuits

🔲 Section 2: Digital Logic

1. Introduction

Combinational circuits are digital logic circuits where the output depends solely on the current input values, with no memory or feedback. The output is a pure function of the present inputs. These circuits form the building blocks of arithmetic logic units, data selectors, and code converters.

2.3 Ripple Carry Adder

An n-bit ripple carry adder chains n full adders, where the carry output of each stage feeds the carry input of the next stage.

flowchart LR subgraph FA0["Full Adder 0"] A0["A0, B0"] --> S0["S0"] end subgraph FA1["Full Adder 1"] A1["A1, B1"] --> S1["S1"] end subgraph FA2["Full Adder 2"] A2["A2, B2"] --> S2["S2"] end subgraph FA3["Full Adder 3"] A3["A3, B3"] --> S3["S3"] end Cin["C_in=0"] --> FA0 FA0 -- "C1" --> FA1 FA1 -- "C2" --> FA2 FA2 -- "C3" --> FA3 FA3 --> Cout["C_out"]

Delay: The worst-case propagation delay of an n-bit ripple carry adder is 2n · t_gate because the carry must ripple through all n stages. This is improved by carry lookahead adders.

2.4 Carry Lookahead Adder (CLA)

CLA computes carries in parallel using generate (G) and propagate (P) signals:

  • Gi = Ai · Bi (generate)
  • Pi = Ai XOR Bi (propagate)
  • Ci+1 = Gi + Pi · Ci

4.3 MUX Tree (Building larger MUX from smaller ones)

An 8:1 MUX can be constructed from two 4:1 MUXes and one 2:1 MUX:

4.4 Implementing Boolean Functions with MUX

Any n-variable Boolean function can be implemented using a 2n-1:1 MUX. Use (n-1) variables as select lines and connect the inputs to 0, 1, the remaining variable, or its complement.

Example: F(A,B,C) = Σm(1,2,6,7) using a 4:1 MUX with A,B as select lines:
S1=A, S0=B: I0=C, I1=C', I2=0, I3=1

10. GATE Exam Tips

  • Know how to implement any Boolean function using MUX or decoder + OR gates.
  • Ripple carry adder delay = 2n gate delays. CLA delay = O(log n).
  • A 1:2n DEMUX is equivalent to an n:2n decoder with enable.
  • Priority encoders handle simultaneous inputs; simple encoders do not.
  • For n-variable function: use 2n-1:1 MUX or n-to-2n decoder.

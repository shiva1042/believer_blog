Section 1: Engineering Mathematics

1. Fundamental Counting Principles

1.1 Addition Principle (Rule of Sum)

If task A can be done in m ways and task B can be done in n ways, and the two tasks cannot be done simultaneously, then A or B can be done in m + n ways.

1.2 Multiplication Principle (Rule of Product)

If task A can be done in m ways and for each way of doing A, task B can be done in n ways, then A followed by B can be done in m × n ways.

Example: A license plate has 2 letters followed by 3 digits.
Number of plates = 26 × 26 × 10 × 10 × 10 = 676,000

1.3 Pigeonhole Principle

If n items are placed into m containers and n > m, then at least one container holds more than one item.

Generalized: If n items are placed into m containers, at least one container holds ⌈n/m⌉ items.
Problem: In a group of 367 people, prove that at least two share the same birthday.
Solution: There are 366 possible birthdays (including Feb 29). By the pigeonhole principle, with 367 people and 366 pigeonholes, at least two people must share a birthday.

TypeFormulaDescription
C(n, r)n! / (r! · (n - r)!)r items from n distinct items

Key identities:
C(n, r) = C(n, n - r)
C(n, r) = C(n - 1, r - 1) + C(n - 1, r)   (Pascal's identity)
∑r=0n C(n, r) = 2n
∑r=0n (-1)r C(n, r) = 0
Problem: How many ways can 10 identical balls be distributed into 4 distinct boxes?
Solution: Stars and bars: C(10 + 4 - 1, 10) = C(13, 10) = C(13, 3) = 286.

3. Inclusion-Exclusion Principle

|A1 ∪ A2 ∪ ... ∪ An| = ∑|Ai| - ∑|Ai ∩ Aj| + ∑|Ai ∩ Aj ∩ Ak| - ... + (-1)n+1|A1 ∩ ... ∩ An|
Problem: How many integers from 1 to 100 are divisible by 2, 3, or 5?
Solution:
|A2| = 50, |A3| = 33, |A5| = 20
|A2 ∩ A3| = 16, |A2 ∩ A5| = 10, |A3 ∩ A5| = 6
|A2 ∩ A3 ∩ A5| = 3
Answer = 50 + 33 + 20 - 16 - 10 - 6 + 3 = 74

3.1 Derangements

A derangement is a permutation where no element appears in its original position.

D(n) = n! ∑k=0n (-1)k / k! = n! [1 - 1/1! + 1/2! - 1/3! + ... + (-1)n/n!]

First few values: D(1) = 0, D(2) = 1, D(3) = 2, D(4) = 9, D(5) = 44.

4. Recurrence Relations

A recurrence relation defines a sequence where each term is expressed in terms of previous terms.

4.1 Linear Homogeneous Recurrence with Constant Coefficients

an = c1an-1 + c2an-2 + ... + ckan-k

4.2 Solving via Characteristic Equation

Replace an with rn to get the characteristic equation:

rk - c1rk-1 - c2rk-2 - ... - ck = 0
RootsGeneral Solution
k distinct roots r1, r2, ..., rkan = A1r1n + A2r2n + ... + Akrkn
Repeated root r with multiplicity man = (A0 + A1n + ... + Am-1nm-1)rn
Problem: Solve an = 5an-1 - 6an-2 with a0 = 1, a1 = 4.
Solution:
Characteristic equation: r2 - 5r + 6 = 0 ⇒ (r - 2)(r - 3) = 0 ⇒ r = 2, 3
General solution: an = A · 2n + B · 3n
Using initial conditions: a0 = A + B = 1, a1 = 2A + 3B = 4
Solving: A = -1, B = 2
Answer: an = -2n + 2 · 3n = 2 · 3n - 2n

4.3 Non-Homogeneous Recurrence

an = c1an-1 + c2an-2 + ... + f(n)

Solution = Homogeneous solution + Particular solution. The particular solution depends on the form of f(n).

f(n)Particular Solution Form

bnTry K · bn (or Kn · bn if b is a root)

6. Generating Functions

A generating function encodes a sequence {a0, a1, a2, ...} as a formal power series.

6.1 Ordinary Generating Function (OGF)

G(x) = ∑n=0∞ an xn = a0 + a1x + a2x2 + ...

6.2 Common Generating Functions

SequenceOGF
{1, 1, 1, ...}1 / (1 - x)
{1, a, a2, ...}1 / (1 - ax)
{1, 2, 3, 4, ...}1 / (1 - x)2
{C(n,0), C(n,1), ..., C(n,n)}(1 + x)n
{1, 1, 1/2!, 1/3!, ...}ex (EGF)

6.3 Exponential Generating Function (EGF)

E(x) = ∑n=0∞ an xn / n!

EGFs are particularly useful when the sequence involves permutations or labeled objects.

Problem: Find the generating function for the number of ways to select r items from n types with unlimited repetition.
Solution: Each type contributes a factor of (1 + x + x2 + ...) = 1/(1-x).
For n types: G(x) = 1/(1-x)n.
Coefficient of xr in 1/(1-x)n = C(n+r-1, r), confirming the stars and bars formula.

8. Key GATE Takeaways

  • Master the counting decision tree: ordered vs unordered, with vs without repetition.
  • Pigeonhole principle problems appear frequently as short-answer questions.
  • Practice solving second-order linear recurrences using the characteristic equation.
  • Know the closed forms for Fibonacci, Catalan numbers, and Tower of Hanoi.
  • Inclusion-exclusion is essential for derangement and surjection counting.
  • Generating functions are powerful for solving recurrences and counting problems.
  • Stars and bars: distributing r identical objects into n distinct boxes = C(n+r-1, r).

Context-Free Languages and Pushdown Automata

🔄 Section 6: Theory of Computation

1. Context-Free Grammars (CFG)

1.1 Formal Definition

A CFG is a 4-tuple G = (V, T, P, S) where:

  • V: Finite set of variables (non-terminals)
  • T: Finite set of terminals (alphabet symbols)
  • P: Finite set of production rules of the form A -> alpha, where A is in V and alpha is in (V union T)*
  • S: Start variable (S in V)

1.2 Derivations

Leftmost Derivation: At each step, replace the leftmost non-terminal.

Rightmost Derivation: At each step, replace the rightmost non-terminal.

Example CFG for L = {a^n b^n | n >= 1}:
  S -> aSb | ab

Derivation of "aaabbb":
  S => aSb => aaSbb => aaabbb  (leftmost derivation)

1.3 Parse Trees

A parse tree is a graphical representation of a derivation. The root is the start symbol, leaves are terminals, and internal nodes are non-terminals. The yield (concatenation of leaves) gives the derived string.

2. Ambiguity in Grammars

A grammar is ambiguous if there exists a string that has two or more distinct leftmost derivations (equivalently, two or more distinct parse trees).

Ambiguous grammar for arithmetic:
  E -> E + E | E * E | id

String "id + id * id" has two parse trees:
  (id + id) * id   and   id + (id * id)

Resolving Ambiguity

Rewrite the grammar to enforce precedence and associativity:

Unambiguous grammar:
  E -> E + T | T
  T -> T * F | F
  F -> id | (E)
Important: Some context-free languages are inherently ambiguous — every grammar for them is ambiguous. Example: L = {a^i b^j c^k | i = j or j = k}.

3. Normal Forms

3.1 Chomsky Normal Form (CNF)

Every production is in one of these forms:

  • A -> BC (two non-terminals)
  • A -> a (single terminal)
  • S -> epsilon (only if epsilon is in the language, and S does not appear on any RHS)

Conversion to CNF

  1. Eliminate epsilon-productions (A -> epsilon).
  2. Eliminate unit productions (A -> B).
  3. Eliminate useless symbols.
  4. Replace terminals in bodies of length >= 2 with new non-terminals.
  5. Break bodies of length > 2 into chains of binary productions.
GATE Tip: CNF is important because the CYK parsing algorithm requires the grammar to be in CNF. CYK runs in O(n3|G|) time where n is the string length.

3.2 Greibach Normal Form (GNF)

Every production is of the form: A -> a alpha, where a is a terminal and alpha is a (possibly empty) string of non-terminals.

GNF ensures that exactly one terminal is generated at each derivation step. Useful for constructing PDAs — each step reads exactly one input symbol.

5. Pushdown Automata (PDA)

5.1 Formal Definition

A PDA is a 7-tuple M = (Q, Sigma, Gamma, delta, q0, Z0, F) where:

  • Q: Finite set of states
  • Sigma: Input alphabet
  • Gamma: Stack alphabet
  • delta: Transition function, delta: Q x (Sigma union {epsilon}) x Gamma -> P(Q x Gamma*)
  • q0: Start state
  • Z0: Initial stack symbol
  • F: Set of accept states

5.2 Acceptance Modes

ModeConditionEquivalence
By Final StateInput consumed and in accept stateBoth modes are equivalent in power
By Empty StackInput consumed and stack is empty

5.3 Deterministic PDA (DPDA) vs. Non-deterministic PDA (NPDA)

  • DPDA: At most one move possible from any configuration.
  • NPDA: Multiple moves possible (more powerful than DPDA).
  • DPDA recognizes DCFL (a proper subset of CFL).
  • Every regular language is a DCFL. DCFL is a proper subset of CFL.
  • Example: {ww^R | w in {a,b}*} is CFL but not DCFL.

6. CFG to PDA Conversion

Given a CFG G = (V, T, P, S), construct a PDA that simulates leftmost derivations:

  1. Push the start symbol S onto the stack.
  2. If top of stack is a non-terminal A, non-deterministically choose a production A -> alpha and replace A with alpha on the stack.
  3. If top of stack is a terminal a that matches the current input symbol, pop it and advance the input.
  4. Accept if input is consumed and stack is empty.

7. Pumping Lemma for Context-Free Languages

Used to prove a language is NOT context-free.

Statement

If L is context-free, then there exists a pumping length p such that for every string s in L with |s| >= p, s can be written as s = uvwxy where:

  1. |vx| > 0 (at least one of v, x is non-empty)
  2. |vwx| ≤ p
  3. For all i >= 0, uv^i wx^i y is in L

Example: L = {a^n b^n c^n | n >= 0} is not context-free

Choose s = a^p b^p c^p.
For any decomposition s = uvwxy with |vwx| ≤ p and |vx| > 0:
  vwx cannot contain all three symbols (a, b, c) since |vwx| ≤ p.
  So pumping (i=2) will increase the count of at most two of
  the three symbols, making the string not in L.
Contradiction. L is not context-free.

10. Key GATE Takeaways

  • CFGs are more powerful than regular grammars and can express nested structures (balanced parentheses, a^n b^n).
  • Ambiguity means multiple parse trees for the same string. Some CFLs are inherently ambiguous.
  • CNF is required for CYK parsing. Every CFG can be converted to CNF.
  • PDA = CFG in language recognition power. DPDA recognizes a proper subset (DCFLs).
  • CFLs are closed under union, concatenation, Kleene star, but NOT intersection or complement.
  • CFL intersect Regular = CFL. This is a powerful tool for proofs.
  • Pumping lemma for CFLs uses two pumpable substrings (v and x).

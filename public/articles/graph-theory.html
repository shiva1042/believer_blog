Graph Theory

Section 1: Engineering Mathematics

1. Basic Definitions

A graph G = (V, E) consists of a set of vertices (nodes) V and a set of edges E connecting pairs of vertices.

TermDefinition
Degree of vertexNumber of edges incident to the vertex (deg(v))
Adjacent verticesTwo vertices connected by an edge
PathSequence of distinct vertices connected by edges
CyclePath that starts and ends at the same vertex
Simple graphNo self-loops or multiple edges
Connected graphPath exists between every pair of vertices
Handshaking Theorem: ∑ deg(v) = 2|E| for all v in V
Corollary: Number of vertices with odd degree is always even.

2. Types of Graphs

2.1 Undirected Graph Example

graph LR A --- B B --- C C --- D D --- A A --- C B --- D

This is a complete graph K4 with 4 vertices and 6 edges. Every vertex has degree 3.

2.2 Directed Graph (Digraph) Example

graph LR A -->|5| B B -->|3| C C -->|7| D D -->|2| A A -->|4| C

A weighted directed graph where edges have directions and weights. In-degree and out-degree are defined separately.

2.3 Bipartite Graph Example

graph LR subgraph Set X X1 X2 X3 end subgraph Set Y Y1 Y2 Y3 end X1 --- Y1 X1 --- Y2 X2 --- Y2 X2 --- Y3 X3 --- Y1 X3 --- Y3

A bipartite graph has vertices partitioned into two sets X and Y, with edges only between sets, never within. A graph is bipartite iff it contains no odd-length cycle.

3. Connectivity

3.1 Connected Components

A connected component is a maximal connected subgraph. An undirected graph with n vertices and k connected components has at least n - k edges.

3.2 Cut Vertices and Bridges

  • Cut vertex (articulation point): A vertex whose removal disconnects the graph.
  • Bridge: An edge whose removal disconnects the graph.
  • A connected graph with no cut vertex is called biconnected.

3.3 Strong and Weak Connectivity (Digraphs)

TypeDefinition
Strongly connectedDirected path exists from u to v and v to u for all pairs u, v
Weakly connectedConnected when direction of edges is ignored
Strongly connected componentMaximal strongly connected subgraph

4. Euler Paths and Circuits

An Euler path visits every edge exactly once. An Euler circuit is an Euler path that starts and ends at the same vertex.

TypeUndirected Graph ConditionDirected Graph Condition
Euler CircuitConnected, all vertices have even degreeStrongly connected, in-degree = out-degree for all
Euler PathConnected, exactly 0 or 2 vertices with odd degreeWeakly connected, at most one vertex has out - in = 1, at most one has in - out = 1, rest equal
Example: K4 has all vertices of degree 3 (odd), so it does NOT have an Euler circuit. But K3 has all vertices of degree 2 (even), so it HAS an Euler circuit.

5. Hamilton Paths and Circuits

A Hamilton path visits every vertex exactly once. A Hamilton circuit visits every vertex exactly once and returns to the start.

5.1 Sufficient Conditions (not necessary)

  • Dirac's Theorem: If deg(v) ≥ n/2 for all vertices in a simple graph with n ≥ 3, then G has a Hamiltonian circuit.
  • Ore's Theorem: If deg(u) + deg(v) ≥ n for every pair of non-adjacent vertices u, v, then G has a Hamiltonian circuit.

Key difference from Euler: No simple necessary and sufficient condition exists for Hamilton paths/circuits. The problem of determining if a Hamiltonian path exists is NP-complete.

6. Graph Coloring

A proper k-coloring assigns one of k colors to each vertex such that no two adjacent vertices share a color. The chromatic number χ(G) is the minimum k for which G has a proper k-coloring.

GraphChromatic Number

Cn (n even)χ = 2
Cn (n odd)χ = 3
Bipartite graphχ = 2

Planar graphχ ≤ 4 (Four Color Theorem)

χ(G) ≥ n / α(G) where α(G) is the independence number

7. Matching

A matching in a graph is a set of edges with no shared vertices.

7.1 Key Definitions

TermDefinition
Maximum matchingMatching with the largest possible number of edges
Perfect matchingMatching that covers all vertices
Maximal matchingMatching that cannot be extended by adding an edge

7.2 Hall's Marriage Theorem

A bipartite graph G = (X ∪ Y, E) has a matching that saturates every vertex in X if and only if for every subset S ⊆ X:

|N(S)| ≥ |S| (Hall's condition)

where N(S) is the set of vertices in Y adjacent to at least one vertex in S.

8. Planar Graphs and Euler's Formula

Euler's Formula: V - E + F = 2 (for connected planar graphs)
where V = vertices, E = edges, F = faces (including the outer face)

8.1 Consequences

  • For simple connected planar graph (V ≥ 3): E ≤ 3V - 6
  • For bipartite planar graph (V ≥ 3): E ≤ 2V - 4
  • K5 is not planar (10 edges > 3(5) - 6 = 9)
  • K3,3 is not planar (9 edges > 2(6) - 4 = 8)
  • Kuratowski's Theorem: G is planar iff it contains no subdivision of K5 or K3,3

9. Trees

9.1 Equivalent Definitions

For a graph G with n vertices, any two of the following imply the third (and that G is a tree):

  • G is connected
  • G is acyclic
  • G has n - 1 edges

10. Key GATE Takeaways

  • Master the Handshaking theorem and its applications in degree calculations.
  • Know Euler path/circuit conditions for both directed and undirected graphs.
  • Chromatic number questions are very common: know values for standard graphs.
  • Euler's formula V - E + F = 2 and its corollaries for planarity testing.
  • Trees: n vertices means n - 1 edges, Cayley's formula for labeled trees.
  • Bipartite iff no odd cycle. Hall's theorem for perfect matchings.
  • Hamilton path existence is NP-complete; know Dirac's and Ore's sufficient conditions.

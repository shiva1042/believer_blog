1. Introduction

A graph G = (V, E) consists of a set of vertices V and a set of edges E. Graphs model relationships between objects and are fundamental to computer science algorithms.

Space tip: For an undirected graph, the adjacency matrix is symmetric. For a directed graph, it is not. Adjacency list is preferred when E = O(V), which is typical in practice.

3. Breadth-First Search (BFS)

BFS explores all vertices at the current depth level before moving to vertices at the next depth level. It uses a queue.

3.1 Algorithm

void BFS(int graph[][], int V, int source) {
    bool visited[V] = {false};
    Queue q;
    visited[source] = true;
    enqueue(&q, source);

    while (!isEmpty(&q)) {
        int u = dequeue(&q);
        printf("%d ", u);
        for (int v = 0; v < V; v++) {
            if (graph[u][v] && !visited[v]) {
                visited[v] = true;
                enqueue(&q, v);
            }
        }
    }
}

4. Depth-First Search (DFS)

DFS explores as far as possible along each branch before backtracking. It uses a stack (or recursion).

4.1 Algorithm

void DFS(int graph[][], int V, int u, bool visited[]) {
    visited[u] = true;
    printf("%d ", u);
    for (int v = 0; v < V; v++) {
        if (graph[u][v] && !visited[v])
            DFS(graph, V, v, visited);
    }
}

Cycle detection: A directed graph has a cycle if and only if DFS finds a back edge. For undirected graphs, any edge to a visited vertex (that is not the parent) indicates a cycle.

4.5 DFS Applications

  • Cycle detection
  • Topological sorting
  • Strongly connected components (Kosaraju's, Tarjan's)
  • Path finding
  • Articulation points and bridges

5. Topological Sort

A topological sort of a DAG is a linear ordering of vertices such that for every directed edge (u, v), u comes before v. It is only possible on DAGs.

5.1 DFS-Based Algorithm

void topologicalSortUtil(int u, bool visited[], Stack *stack, int graph[][], int V) {
    visited[u] = true;
    for (int v = 0; v < V; v++) {
        if (graph[u][v] && !visited[v])
            topologicalSortUtil(v, visited, stack, graph, V);
    }
    push(stack, u);  // push after all descendants are processed
}

// Pop stack to get topological order

5.2 Kahn's Algorithm (BFS-Based)

// 1. Compute in-degree of all vertices
// 2. Enqueue all vertices with in-degree 0
// 3. While queue is not empty:
//    a. Dequeue vertex u, add to result
//    b. For each neighbor v of u:
//       - Decrease in-degree of v
//       - If in-degree of v becomes 0, enqueue v
// 4. If result has all V vertices, it's a valid topological order
//    Otherwise, graph has a cycle

5.3 Topological Sort Example

Possible topological orders: A, B, C, D, E or A, B, D, C, E or B, A, C, D, E or B, A, D, C, E

Key fact: Topological sort is not unique unless the graph is a Hamiltonian path (a path that visits every vertex exactly once).

6. Connected Components

6.1 Undirected Graphs

A connected component is a maximal set of vertices such that there is a path between every pair. Use BFS or DFS from each unvisited vertex to find all components.

int countComponents(int graph[][], int V) {
    bool visited[V] = {false};
    int count = 0;
    for (int i = 0; i < V; i++) {
        if (!visited[i]) {
            DFS(graph, V, i, visited);
            count++;
        }
    }
    return count;
}

6.2 Strongly Connected Components (Directed Graphs)

In a directed graph, a strongly connected component (SCC) is a maximal set of vertices where every vertex is reachable from every other vertex in the set.

AlgorithmApproachTime

9. GATE Exam Tips

  • BFS and DFS both have O(V + E) time with adjacency list, O(V^2) with adjacency matrix.
  • BFS gives shortest path in unweighted graphs; use Dijkstra for weighted graphs.
  • A directed graph has a cycle iff DFS finds a back edge.
  • Topological sort exists only for DAGs. If Kahn's algorithm does not include all vertices, the graph has a cycle.
  • For an undirected connected graph: E >= V - 1. If E = V - 1, it is a tree.
  • Number of connected components using DFS/BFS: count the number of DFS/BFS calls from the outer loop.
  • Adjacency matrix is symmetric for undirected graphs; not necessarily for directed graphs.
  • Handshaking lemma: sum of all degrees = 2|E| (undirected graph).

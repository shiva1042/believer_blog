2. Definitions and Properties

2.1 Groupoid (Magma)

A set S with a binary operation * is a groupoid if * is closed on S: for all a, b ∈ S, a * b ∈ S.

2.2 Semigroup

A groupoid (S, *) is a semigroup if * is associative: for all a, b, c ∈ S, (a * b) * c = a * (b * c).

Example: (N, +) is a semigroup. Addition on natural numbers is closed and associative. Similarly, (strings, concatenation) is a semigroup.

2.3 Monoid

A semigroup (S, *) is a monoid if there exists an identity element e ∈ S such that for all a ∈ S: e * a = a * e = a.

Examples of monoids:
  • (N, +) with identity 0
  • (N, ×) with identity 1
  • (strings, concatenation) with identity "" (empty string)
  • (P(S), ∪) with identity ∅
  • (P(S), ∩) with identity S

2.4 Group

A monoid (G, *) is a group if every element has an inverse: for all a ∈ G, there exists a−1 ∈ G such that a * a−1 = a−1 * a = e.

PropertyRequirement
Closure∀a, b ∈ G: a * b ∈ G
Associativity∀a, b, c ∈ G: (a * b) * c = a * (b * c)
Identity∃e ∈ G: ∀a ∈ G, e * a = a * e = a
Inverse∀a ∈ G, ∃a−1 ∈ G: a * a−1 = a−1 * a = e
Examples of groups:
  • (Z, +) — integers under addition (identity: 0, inverse of a: -a)
  • (Q \ {0}, ×) — nonzero rationals under multiplication (identity: 1, inverse of a: 1/a)
  • (Zn, +n) — integers modulo n under addition mod n
Non-examples:
  • (N, +) is NOT a group (no inverses: there is no natural number n such that 3 + n = 0)
  • (Z, ×) is NOT a group (2 has no integer multiplicative inverse)

3. Important Group Properties

3.1 Uniqueness Results

  • The identity element of a group is unique.
  • The inverse of each element is unique.
  • (a−1)−1 = a
  • (a * b)−1 = b−1 * a−1 (socks-shoes property)

3.2 Cancellation Laws

In any group: if a * b = a * c, then b = c (left cancellation), and if b * a = c * a, then b = c (right cancellation).

4. Abelian Groups

A group (G, *) is Abelian (commutative) if for all a, b ∈ G: a * b = b * a.

Examples: (Z, +), (Q \ {0}, ×), (Zn, +n) are all Abelian.
Non-example: The group of 2×2 invertible matrices under multiplication is non-Abelian (matrix multiplication is not commutative in general).

5. Order of an Element and Group

Order of group |G|: the number of elements in G
Order of element a (denoted ord(a)): the smallest positive integer n such that an = e

If no such n exists, the element has infinite order.

Example: In (Z6, +6) = {0, 1, 2, 3, 4, 5}:
ord(0) = 1 (0 is the identity)
ord(1) = 6 (1+1+1+1+1+1 = 6 ≡ 0 mod 6)
ord(2) = 3 (2+2+2 = 6 ≡ 0 mod 6)
ord(3) = 2 (3+3 = 6 ≡ 0 mod 6)
ord(4) = 3 (4+4+4 = 12 ≡ 0 mod 6)
ord(5) = 6 (5+5+5+5+5+5 = 30 ≡ 0 mod 6)

6. Cyclic Groups

A group G is cyclic if there exists an element g ∈ G (called a generator) such that every element of G can be written as gn for some integer n. We write G = ⟨g⟩.

6.1 Key Properties of Cyclic Groups

  • Every cyclic group is Abelian.
  • A cyclic group of order n is isomorphic to (Zn, +n).
  • An infinite cyclic group is isomorphic to (Z, +).
  • An element a generates Zn iff GCD(a, n) = 1.
  • Number of generators of Zn = φ(n) (Euler's totient).
  • Every subgroup of a cyclic group is cyclic.
  • For each divisor d of n, there is exactly one subgroup of Zn with order d.
Example: Z6 = ⟨1⟩ = ⟨5⟩ (generators are 1 and 5 since GCD(1,6)=1 and GCD(5,6)=1).
φ(6) = 6(1 − 1/2)(1 − 1/3) = 2 generators.

7. Subgroups

A non-empty subset H of group G is a subgroup if H itself is a group under the same operation. We write H ≤ G.

7.1 Subgroup Test

H ⊆ G is a subgroup iff for all a, b ∈ H: a * b−1 ∈ H (one-step subgroup test).

7.2 Trivial Subgroups

Every group G has at least two subgroups: {e} (trivial subgroup) and G itself (improper subgroup).

8. Lagrange's Theorem

If H is a subgroup of a finite group G, then |H| divides |G|.
Equivalently: |G| = |H| · [G : H], where [G : H] is the index (number of cosets).

8.1 Consequences of Lagrange's Theorem

  • The order of every element divides the order of the group: ord(a) | |G|.
  • A group of prime order p has no proper non-trivial subgroups.
  • A group of prime order is cyclic (and hence Abelian).
  • For any a ∈ G: a|G| = e.
  • Converse is false: If k divides |G|, there need not exist a subgroup of order k (e.g., A4 has order 12 but no subgroup of order 6).
Example: G = Z12 has order 12. Divisors of 12: 1, 2, 3, 4, 6, 12.
Subgroups: {0}, {0,6}, {0,4,8}, {0,3,6,9}, {0,2,4,6,8,10}, Z12
Each subgroup order divides 12, confirming Lagrange's theorem.

9. Cosets

For a subgroup H of G and element a ∈ G:

  • Left coset: aH = {a * h | h ∈ H}
  • Right coset: Ha = {h * a | h ∈ H}

All cosets have the same size |H|, and they partition G into disjoint sets.

10. Group Homomorphisms and Isomorphisms

10.1 Properties of Homomorphisms

  • f(eG) = eH
  • f(a−1) = f(a)−1
  • f is injective iff ker(f) = {eG}
  • Image of a subgroup is a subgroup

12. Key GATE Takeaways

  • Verify group axioms methodically: closure, associativity, identity, inverse.
  • Lagrange's theorem is heavily tested: order of subgroup divides order of group.
  • Groups of prime order are always cyclic (and Abelian).
  • Every cyclic group is Abelian, but not every Abelian group is cyclic.
  • Know how to compute the order of elements in Zn.
  • Number of generators of Zn equals Euler's totient φ(n).

2.2 Cofactor Expansion

det(A) = ∑j=1n aij Cij (expansion along row i)
where Cij = (-1)i+j Mij and Mij is the minor (determinant of submatrix)
Example: For A = [[2, 1, 3], [0, -1, 2], [1, 4, -1]]
Expanding along first row:
det(A) = 2[(-1)(-1) - (2)(4)] - 1[(0)(-1) - (2)(1)] + 3[(0)(4) - (-1)(1)]
= 2[1 - 8] - 1[0 - 2] + 3[0 + 1]
= 2(-7) - 1(-2) + 3(1) = -14 + 2 + 3 = -9

3. Rank of a Matrix

Rank(A) = number of non-zero rows in row echelon form
= number of linearly independent rows = number of linearly independent columns

3.1 Rank Properties

  • rank(A) ≤ min(m, n) for m × n matrix
  • rank(A) = rank(AT)
  • rank(AB) ≤ min(rank(A), rank(B))
  • rank(A + B) ≤ rank(A) + rank(B)
  • Nullity(A) = n - rank(A) (Rank-Nullity Theorem)
  • A is invertible iff rank(A) = n (for n × n matrix)

4. System of Linear Equations (Ax = b)

4.1 Consistency (Rouche-Capelli Theorem)

ConditionResult
rank(A) ≠ rank([A|b])No solution (inconsistent)
rank(A) = rank([A|b]) = nUnique solution
rank(A) = rank([A|b]) < nInfinitely many solutions

For a homogeneous system Ax = 0: always consistent (x = 0 is always a solution). Has non-trivial solutions iff rank(A) < n.

4.2 Gaussian Elimination

Transform [A|b] to row echelon form using elementary row operations: (1) swap rows, (2) multiply row by nonzero scalar, (3) add multiple of one row to another. Then back-substitute.

Example: Solve the system: x + y + z = 6, 2x + 3y + z = 14, x + 2y + 3z = 14
Augmented matrix: [[1,1,1,6], [2,3,1,14], [1,2,3,14]]
R2 ← R2 - 2R1: [[1,1,1,6], [0,1,-1,2], [1,2,3,14]]
R3 ← R3 - R1: [[1,1,1,6], [0,1,-1,2], [0,1,2,8]]
R3 ← R3 - R2: [[1,1,1,6], [0,1,-1,2], [0,0,3,6]]
Back-substitution: z = 2, y = 2 + z = 4, x = 6 - y - z = 0
Solution: (x, y, z) = (0, 4, 2)

4.3 Cramer's Rule

For Ax = b where det(A) ≠ 0:
xi = det(Ai) / det(A)
where Ai is the matrix A with column i replaced by b.

5. Eigenvalues and Eigenvectors

For a square matrix A, scalar λ is an eigenvalue and non-zero vector v is an eigenvector if:

Av = λv    equivalently    (A - λI)v = 0

5.1 Finding Eigenvalues

Solve the characteristic equation: det(A - λI) = 0

Example: A = [[4, 1], [2, 3]]
det(A - λI) = (4 - λ)(3 - λ) - 2 = λ2 - 7λ + 10 = (λ - 5)(λ - 2) = 0
Eigenvalues: λ1 = 5, λ2 = 2

For λ = 5: (A - 5I)v = 0 ⇒ [[-1,1],[2,-2]]v = 0 ⇒ v = t[1,1]T
For λ = 2: (A - 2I)v = 0 ⇒ [[2,1],[2,1]]v = 0 ⇒ v = t[-1,2]T

5.3 Cayley-Hamilton Theorem

Every square matrix satisfies its own characteristic equation. If the characteristic polynomial is p(λ) = λ2 - 7λ + 10, then p(A) = A2 - 7A + 10I = 0.

This is useful for computing A-1 and higher powers of A.

6. LU Decomposition

LU decomposition factors a matrix A into a product of a lower triangular matrix L and an upper triangular matrix U: A = LU.

Example: Decompose A = [[2, 1, 1], [4, 3, 3], [8, 7, 9]]

Perform Gaussian elimination tracking multipliers:
R2 ← R2 - 2R1 (multiplier = 2): U = [[2,1,1], [0,1,1], [8,7,9]]
R3 ← R3 - 4R1 (multiplier = 4): U = [[2,1,1], [0,1,1], [0,3,5]]
R3 ← R3 - 3R2 (multiplier = 3): U = [[2,1,1], [0,1,1], [0,0,2]]

L = [[1,0,0], [2,1,0], [4,3,1]], U = [[2,1,1], [0,1,1], [0,0,2]]

To solve Ax = b: First solve Ly = b (forward substitution), then Ux = y (back substitution).

7. Diagonalization

A is diagonalizable iff A has n linearly independent eigenvectors.
A = PDP-1 where D = diag(λ1, ..., λn) and P = [v1 | v2 | ... | vn]
Ak = PDkP-1

7.1 Sufficient Conditions for Diagonalizability

  • A has n distinct eigenvalues (always diagonalizable)
  • A is symmetric (always diagonalizable, with orthogonal P)
  • For each eigenvalue: geometric multiplicity = algebraic multiplicity

9. Key GATE Takeaways

  • Know matrix types and their properties (symmetric, orthogonal, idempotent, etc.).
  • det(kA) = kndet(A) is a commonly tested trap.
  • Rank-nullity theorem: rank(A) + nullity(A) = number of columns.
  • For Ax = b: check rank(A) vs rank([A|b]) for consistency.
  • Sum of eigenvalues = trace, product = determinant.
  • Cayley-Hamilton theorem for computing inverse and matrix powers.
  • Eigenvalue properties for special matrices (symmetric, orthogonal, etc.) save computation time.

Linked Lists

💻 Section 4: Programming and Data Structures

1. Introduction

A linked list is a linear data structure where elements (nodes) are stored in non-contiguous memory locations. Each node contains data and one or more pointers to other nodes. Unlike arrays, linked lists allow efficient insertion and deletion without shifting elements.

2. Singly Linked List

2.1 Structure

flowchart LR HEAD["HEAD"] --> N1["10 | next"] N1 --> N2["20 | next"] N2 --> N3["30 | next"] N3 --> N4["40 | NULL"] 
struct Node {
    int data;
    struct Node *next;
};

// Create a new node
struct Node* createNode(int data) {
    struct Node* node = (struct Node*)malloc(sizeof(struct Node));
    node->data = data;
    node->next = NULL;
    return node;
}

2.2 Insertion Operations

Insert at Beginning — O(1)

void insertAtHead(struct Node **head, int data) {
    struct Node *newNode = createNode(data);
    newNode->next = *head;
    *head = newNode;
}

Insert at End — O(n)

void insertAtTail(struct Node **head, int data) {
    struct Node *newNode = createNode(data);
    if (*head == NULL) { *head = newNode; return; }
    struct Node *temp = *head;
    while (temp->next != NULL)
        temp = temp->next;
    temp->next = newNode;
}

Insert After a Given Node — O(1) if node is known

void insertAfter(struct Node *prevNode, int data) {
    if (prevNode == NULL) return;
    struct Node *newNode = createNode(data);
    newNode->next = prevNode->next;
    prevNode->next = newNode;
}

2.3 Deletion Operations

Delete by Value — O(n)

void deleteNode(struct Node **head, int key) {
    struct Node *temp = *head, *prev = NULL;
    if (temp != NULL && temp->data == key) {
        *head = temp->next;
        free(temp);
        return;
    }
    while (temp != NULL && temp->data != key) {
        prev = temp;
        temp = temp->next;
    }
    if (temp == NULL) return;
    prev->next = temp->next;
    free(temp);
}

2.4 Reversal of Singly Linked List — O(n)

struct Node* reverse(struct Node *head) {
    struct Node *prev = NULL, *curr = head, *next = NULL;
    while (curr != NULL) {
        next = curr->next;   // save next
        curr->next = prev;   // reverse link
        prev = curr;          // advance prev
        curr = next;          // advance curr
    }
    return prev;  // new head
}

2.5 Recursive Reversal

struct Node* reverseRecursive(struct Node *head) {
    if (head == NULL || head->next == NULL)
        return head;
    struct Node *rest = reverseRecursive(head->next);
    head->next->next = head;
    head->next = NULL;
    return rest;
}

3.3 Insertion in Doubly Linked List

void insertAfterDLL(struct DNode *prevNode, int data) {
    struct DNode *newNode = createDNode(data);
    newNode->next = prevNode->next;
    newNode->prev = prevNode;
    if (prevNode->next != NULL)
        prevNode->next->prev = newNode;
    prevNode->next = newNode;
}

4. Circular Linked List

4.1 Singly Circular Linked List

The last node points back to the first node instead of NULL.

flowchart LR N1["10"] --> N2["20"] N2 --> N3["30"] N3 --> N4["40"] N4 --> N1

4.2 Doubly Circular Linked List

Last node's next points to first; first node's prev points to last.

4.3 Applications of Circular Lists

  • Round-robin scheduling in operating systems
  • Circular buffers
  • Implementation of queues (single pointer to tail gives O(1) access to both ends)
  • Josephus problem

5. Skip List

A skip list is a probabilistic data structure that uses multiple levels of linked lists to achieve O(log n) average search time.

Skip List vs BST: Skip lists provide similar average performance to balanced BSTs but are simpler to implement, especially for concurrent data structures. The level of each node is determined randomly (typically with probability 1/2).

6. Common Linked List Problems

6.1 Detect Cycle (Floyd's Algorithm)

bool hasCycle(struct Node *head) {
    struct Node *slow = head, *fast = head;
    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) return true;
    }
    return false;
}

6.2 Find Middle Element

struct Node* findMiddle(struct Node *head) {
    struct Node *slow = head, *fast = head;
    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;  // slow is at middle
}

6.3 Merge Two Sorted Lists

struct Node* mergeSorted(struct Node *a, struct Node *b) {
    if (!a) return b;
    if (!b) return a;
    if (a->data <= b->data) {
        a->next = mergeSorted(a->next, b);
        return a;
    } else {
        b->next = mergeSorted(a, b->next);
        return b;
    }
}

8. GATE Exam Tips

  • Reversal of linked list is a very frequently tested topic. Know both iterative and recursive approaches.
  • Floyd's cycle detection: slow pointer moves 1 step, fast moves 2 steps. They meet inside the cycle if one exists.
  • To find the start of a cycle: after detection, reset one pointer to head and move both one step at a time; they meet at the cycle start.
  • For pointer manipulation questions, draw diagrams to trace each step.
  • Deleting a node given only a pointer to it (not head): copy data from next node to current, delete next node. Does not work for the last node.
  • Circular linked list with a tail pointer gives O(1) for both insert at head and insert at tail.

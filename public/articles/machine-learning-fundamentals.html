3. Linear Regression

3.1 Model

Linear regression models the relationship between input features and a continuous output:

Hypothesis: h(x) = w0 + w1*x1 + w2*x2 + ... + wn*xn
Vector form: h(x) = w^T * x

where w = weight vector, x = feature vector

3.2 Cost Function (Mean Squared Error)

J(w) = (1/2m) * SUM[i=1 to m] (h(x_i) - y_i)^2

where m = number of training examples

3.3 Gradient Descent

Gradient descent iteratively updates weights to minimize the cost function:

Repeat until convergence:
    w_j := w_j - alpha * (dJ/dw_j)
    w_j := w_j - alpha * (1/m) * SUM[i=1 to m] (h(x_i) - y_i) * x_i_j

where alpha = learning rate (step size)
Learning Rate: Too large → overshoots, may diverge. Too small → very slow convergence. Typical starting values: 0.01, 0.001.

3.4 Normal Equation (Closed-form Solution)

w = (X^T * X)^(-1) * X^T * y

No iteration needed, but O(n^3) for matrix inversion.
Prefer gradient descent when n (features) is large.

4. Logistic Regression

4.1 Sigmoid Function

sigma(z) = 1 / (1 + e^(-z))

Properties:
  - Output range: (0, 1) -> interpreted as probability
  - sigma(0) = 0.5
  - sigma'(z) = sigma(z) * (1 - sigma(z))

4.2 Decision Boundary

The model predicts class 1 if h(x) = sigma(wTx) ≥ 0.5, i.e., when wTx ≥ 0.

Linear decision boundary: w0 + w1*x1 + w2*x2 = 0

Example: if w = [-3, 1, 1], boundary is x1 + x2 = 3
  Predict 1 if x1 + x2 >= 3
  Predict 0 if x1 + x2 < 3

4.3 Cost Function (Cross-Entropy Loss)

J(w) = -(1/m) * SUM[ y_i * log(h(x_i)) + (1-y_i) * log(1-h(x_i)) ]

This is convex, so gradient descent finds the global minimum.

5. Decision Trees

5.1 Entropy

H(S) = -SUM[i=1 to c] p_i * log2(p_i)

where p_i = proportion of class i in set S
c = number of classes

Example: S has 9 positive, 5 negative (total 14)
H(S) = -(9/14)*log2(9/14) - (5/14)*log2(5/14)
     = -(0.643)(−0.637) - (0.357)(−1.486)
     = 0.410 + 0.531 = 0.940 bits

5.2 Information Gain

IG(S, A) = H(S) - SUM[v in Values(A)] (|S_v|/|S|) * H(S_v)

Choose attribute A with highest information gain to split on.

5.3 Gini Index

Gini(S) = 1 - SUM[i=1 to c] p_i^2

Example: S has 9 positive, 5 negative
Gini = 1 - (9/14)^2 - (5/14)^2
     = 1 - 0.413 - 0.128 = 0.459

6. Support Vector Machines (SVM)

6.1 Linear SVM

SVM finds the hyperplane that maximizes the margin between two classes.

Decision boundary: w^T * x + b = 0
Margin = 2 / ||w||

Optimization: Minimize (1/2)||w||^2
Subject to: y_i(w^T * x_i + b) >= 1  for all i

Support vectors are data points ON the margin boundaries.

6.2 Kernel Trick

When data is not linearly separable, map to a higher-dimensional space using a kernel function:

7.1 Bias-Variance Decomposition

Expected Error = Bias^2 + Variance + Irreducible Noise

Bias = E[h(x)] - f(x)     (systematic error)
Variance = E[(h(x) - E[h(x)])^2]  (sensitivity to training data)

7.2 Regularization

L2 Regularization (Ridge):
  J(w) = (1/2m) * SUM(h(x_i) - y_i)^2 + lambda * SUM(w_j^2)

L1 Regularization (Lasso):
  J(w) = (1/2m) * SUM(h(x_i) - y_i)^2 + lambda * SUM(|w_j|)

L1 promotes sparsity (feature selection).
L2 shrinks all weights uniformly.

8. Cross-Validation

k-Fold Cross-Validation: Split data into k folds, train on k-1 folds, test on the remaining one. Repeat k times and average the scores.

GATE Tip: Leave-one-out CV is k-fold where k = n (number of samples). It has high variance but low bias.

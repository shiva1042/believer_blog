2. GCD and LCM

2.1 Euclidean Algorithm

The Euclidean algorithm computes GCD(a, b) by repeatedly applying: GCD(a, b) = GCD(b, a mod b) until the remainder is 0.

Example: GCD(252, 105)

GCD(252, 105):
  252 = 2 * 105 + 42     => GCD(105, 42)
  105 = 2 * 42 + 21      => GCD(42, 21)
  42  = 2 * 21 + 0        => GCD = 21

Answer: GCD(252, 105) = 21

2.2 Properties

  • GCD(a, b) * LCM(a, b) = a * b
  • GCD(a, 0) = a
  • GCD(a, 1) = 1
  • GCD(a, b) = GCD(a - b, b) if a > b
  • If GCD(a, b) = 1, then a and b are coprime (relatively prime).

2.3 Extended Euclidean Algorithm

Finds integers x, y such that: ax + by = GCD(a, b)

Example: Find x, y for 252x + 105y = 21

Back-substituting from Euclidean algorithm:
  21 = 105 - 2 * 42
  42 = 252 - 2 * 105

  21 = 105 - 2 * (252 - 2*105)
     = 105 - 2*252 + 4*105
     = 5*105 - 2*252

So x = -2, y = 5.  Check: 252*(-2) + 105*5 = -504 + 525 = 21

3. Modular Arithmetic

3.1 Basic Properties

PropertyExpression
Addition(a + b) mod n = [(a mod n) + (b mod n)] mod n
Multiplication(a * b) mod n = [(a mod n) * (b mod n)] mod n
Subtraction(a - b) mod n = [(a mod n) - (b mod n) + n] mod n
Exponentiationa^k mod n can be computed by repeated squaring
Warning: Division is NOT straightforward in modular arithmetic. You need the modular inverse (see below).

3.2 Modular Inverse

The modular inverse of a modulo n is a number x such that: a * x ≡ 1 (mod n)

It exists if and only if GCD(a, n) = 1 (a and n are coprime).

Example: Find inverse of 3 mod 7.

We need x such that 3x ≡ 1 (mod 7).
Try x = 1: 3*1 = 3 (not 1 mod 7)
Try x = 2: 3*2 = 6 (not 1 mod 7)
Try x = 3: 3*3 = 9 ≡ 2 (mod 7) (no)
Try x = 5: 3*5 = 15 ≡ 1 (mod 7) YES!

Inverse of 3 mod 7 = 5

For large numbers, use the Extended Euclidean Algorithm to find the inverse.

4. Euler's Totient Function

Definition: φ(n) counts the number of integers from 1 to n that are coprime with n.

4.1 Computation Rules

CaseFormulaExample
p is primeφ(p) = p - 1φ(7) = 6
p^k (prime power)φ(p^k) = p^k - p^(k-1) = p^(k-1)(p-1)φ(8) = φ(2^3) = 4
Multiplicative (coprime)φ(mn) = φ(m)φ(n) if GCD(m,n)=1φ(12) = φ(4)φ(3) = 2*2 = 4
General formulaφ(n) = n * ∏(1 - 1/p) for each prime p dividing nφ(12) = 12*(1-1/2)*(1-1/3) = 4

4.2 Euler's Theorem

If GCD(a, n) = 1, then: a^φ(n) ≡ 1 (mod n)

Example: Find 2^100 mod 13.

φ(13) = 12 (since 13 is prime)
By Euler's theorem: 2^12 ≡ 1 (mod 13)
100 = 12 * 8 + 4
2^100 = (2^12)^8 * 2^4 ≡ 1^8 * 16 ≡ 16 mod 13 = 3

Answer: 2^100 mod 13 = 3

5. Fermat's Little Theorem

A special case of Euler's theorem when n is prime:

If p is prime and GCD(a, p) = 1, then: a^(p-1) ≡ 1 (mod p)

Equivalently: a^p ≡ a (mod p) for any integer a.

Example: Find 5^302 mod 7.

By Fermat's little theorem: 5^6 ≡ 1 (mod 7)
302 = 6 * 50 + 2
5^302 = (5^6)^50 * 5^2 ≡ 1 * 25 ≡ 25 mod 7 = 4

Answer: 5^302 mod 7 = 4

5.1 Application: Finding Modular Inverse Using Fermat

If p is prime, the inverse of a mod p is: a^(p-2) mod p

Example: Inverse of 3 mod 7 = 3^5 mod 7 = 243 mod 7 = 5.

6. Chinese Remainder Theorem (CRT)

Statement: Given pairwise coprime moduli m1, m2, ..., mk, the system of congruences:

x ≡ a1 (mod m1)
x ≡ a2 (mod m2)
...
x ≡ ak (mod mk)

has a unique solution modulo M = m1 * m2 * ... * mk.

Example: Solve the system:

x ≡ 2 (mod 3)
x ≡ 3 (mod 5)
x ≡ 2 (mod 7)

Step 1: M = 3 * 5 * 7 = 105
Step 2: M1 = 105/3 = 35, M2 = 105/5 = 21, M3 = 105/7 = 15
Step 3: Find inverses:
  35 * y1 ≡ 1 (mod 3) => 2*y1 ≡ 1 (mod 3) => y1 = 2
  21 * y2 ≡ 1 (mod 5) => 1*y2 ≡ 1 (mod 5) => y2 = 1
  15 * y3 ≡ 1 (mod 7) => 1*y3 ≡ 1 (mod 7) => y3 = 1
Step 4: x = a1*M1*y1 + a2*M2*y2 + a3*M3*y3
         = 2*35*2 + 3*21*1 + 2*15*1
         = 140 + 63 + 30 = 233
Step 5: x = 233 mod 105 = 23

Answer: x = 23 (and x = 23 + 105k for all integers k)

Verify: 23 mod 3 = 2, 23 mod 5 = 3, 23 mod 7 = 2. All conditions satisfied.

7. Applications in GATE

  • Cryptography questions: RSA algorithm relies on Euler's theorem and modular arithmetic.
  • Hashing: Hash functions often use modular arithmetic (h(k) = k mod m).
  • Counting problems: GCD/LCM used in combinatorics and scheduling problems.
  • Finding remainders: Fermat and Euler theorems simplify large exponentiation mod problems.
  • Group theory connections: φ(n) is the order of the group (Z/nZ)*.

8. Practice Problems

Problem 1: Find GCD(48, 18) using the Euclidean algorithm.

48 = 2*18 + 12
18 = 1*12 + 6
12 = 2*6 + 0
GCD = 6

Problem 2: Compute φ(36).

36 = 2^2 * 3^2
φ(36) = 36 * (1 - 1/2) * (1 - 1/3) = 36 * 1/2 * 2/3 = 12

Problem 3: Find 7^222 mod 11.

11 is prime, so by Fermat: 7^10 ≡ 1 (mod 11)
222 = 10*22 + 2
7^222 ≡ 7^2 = 49 ≡ 5 (mod 11)
Answer: 5

Problem 4: Find the modular inverse of 5 mod 11.

By Fermat: inverse = 5^(11-2) mod 11 = 5^9 mod 11
5^2 = 25 ≡ 3 (mod 11)
5^4 ≡ 9 (mod 11)
5^8 ≡ 81 ≡ 4 (mod 11)
5^9 = 5^8 * 5 ≡ 4*5 = 20 ≡ 9 (mod 11)
Answer: 9.  Check: 5*9 = 45 ≡ 1 (mod 11)

Problem 5: How many integers from 1 to 100 are coprime with 100?

100 = 2^2 * 5^2
φ(100) = 100 * (1 - 1/2) * (1 - 1/5) = 100 * 1/2 * 4/5 = 40
Answer: 40

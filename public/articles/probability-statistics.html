Section 1: Engineering Mathematics

1. Basic Probability

1.1 Axioms of Probability

  • For any event A: 0 ≤ P(A) ≤ 1
  • P(S) = 1 where S is the sample space
  • For mutually exclusive events A and B: P(A ∪ B) = P(A) + P(B)

1.2 Fundamental Formulas

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
P(A') = 1 - P(A)
P(A ∩ B) = P(A) · P(B|A) = P(B) · P(A|B)

2. Conditional Probability and Independence

2.1 Conditional Probability

P(A|B) = P(A ∩ B) / P(B), provided P(B) > 0

2.2 Independence

Events A and B are independent if and only if P(A ∩ B) = P(A) · P(B), which equivalently means P(A|B) = P(A).

3. Bayes' Theorem

P(Ai|B) = P(B|Ai) · P(Ai) / ∑j P(B|Aj) · P(Aj)

Bayes' Theorem Example (Medical Test):
A disease affects 1% of the population. A test has 95% sensitivity (true positive rate) and 95% specificity (true negative rate).

Question: If a person tests positive, what is the probability they actually have the disease?

P(Disease) = 0.01, P(No Disease) = 0.99
P(+|Disease) = 0.95, P(+|No Disease) = 0.05

P(+) = P(+|Disease) · P(Disease) + P(+|No Disease) · P(No Disease)
= 0.95 × 0.01 + 0.05 × 0.99 = 0.0095 + 0.0495 = 0.059

P(Disease|+) = 0.0095 / 0.059 = 0.161 (about 16.1%)

Despite the high accuracy of the test, a positive result only indicates a 16.1% chance of disease, due to the low base rate. This is a classic example of the base rate fallacy.

7. Normal Distribution: Important Values

For Z ~ N(0, 1) (standard normal):
P(-1 ≤ Z ≤ 1) ≈ 68.27%
P(-2 ≤ Z ≤ 2) ≈ 95.45%
P(-3 ≤ Z ≤ 3) ≈ 99.73%

Standardization: If X ~ N(μ, σ2), then Z = (X - μ)/σ ~ N(0, 1).

8. Joint Distributions and Covariance

Cov(X, Y) = E[XY] - E[X]E[Y]
Correlation: ρ(X, Y) = Cov(X, Y) / (σX · σY), where -1 ≤ ρ ≤ 1
Var(X + Y) = Var(X) + Var(Y) + 2Cov(X, Y)
If X, Y independent: Cov(X, Y) = 0 (converse not always true)

9. Law of Total Probability

P(B) = ∑i P(B|Ai) · P(Ai)
where {A1, A2, ..., An} is a partition of the sample space.

This is the denominator in Bayes' theorem and is essential for computing marginal probabilities.

10. Markov and Chebyshev Inequalities

InequalityStatementCondition
Markov's P(X ≥ a) ≤ E[X]/a X ≥ 0, a > 0
Chebyshev's P(|X - μ| ≥ kσ) ≤ 1/k2 Any distribution with finite variance
Chebyshev Example: If μ = 100 and σ = 15, what is the upper bound for P(|X - 100| ≥ 30)?
k = 30/15 = 2. P(|X - 100| ≥ 30) ≤ 1/4 = 0.25.
At least 75% of data lies within 2 standard deviations of the mean.

  • Master Bayes' theorem -- it appears in GATE almost every year.
  • Know the mean and variance of all standard distributions by heart.
  • Poisson is the only common distribution where mean equals variance.
  • Memoryless property: only Geometric (discrete) and Exponential (continuous).
  • Independent events: P(A ∩ B) = P(A)P(B). Mutually exclusive: P(A ∩ B) = 0. These are different.
  • Chebyshev's inequality gives distribution-free bounds on probability.
  • Central Limit Theorem: sum of many i.i.d. variables approaches Normal distribution.
  • Standardize Normal distributions to use Z-tables: Z = (X - μ)/σ.

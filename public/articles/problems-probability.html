Problem 1: Bayes' Theorem — Disease Testing

Problem: A disease affects 1% of a population. A test has 99% sensitivity (true positive rate) and 95% specificity (true negative rate). If a person tests positive, what is the probability they actually have the disease?
Approach: Use Bayes' theorem: P(D|+) = P(+|D)P(D) / [P(+|D)P(D) + P(+|D')P(D')].
Step-by-step Solution:
Given: P(D) = 0.01, P(D') = 0.99, P(+|D) = 0.99, P(-|D') = 0.95 so P(+|D') = 0.05

Probability tree:
BranchP(Disease)P(Test|Disease)Joint
D and +0.010.990.0099
D and -0.010.010.0001
D' and +0.990.050.0495
D' and -0.990.950.9405
P(+) = 0.0099 + 0.0495 = 0.0594

P(D|+) = 0.0099 / 0.0594 = 0.1667 (approximately 16.67%)
Key Takeaway: Even with a highly accurate test, low prevalence leads to many false positives. This is the "base rate fallacy." In GATE, always compute P(+) using the law of total probability before applying Bayes'.

Problem 2: Conditional Probability — Drawing Cards

Problem: Two cards are drawn without replacement from a standard 52-card deck. Given that the first card is a King, what is the probability that the second card is also a King?
Approach: Use conditional probability directly. After removing one King, 3 Kings remain out of 51 cards.
Step-by-step Solution:
P(2nd is King | 1st is King) = 3/51 = 1/17

Verification using joint probability:
P(both Kings) = (4/52)(3/51) = 12/2652 = 1/221
P(1st is King) = 4/52 = 1/13
P(2nd King | 1st King) = (1/221)/(1/13) = 13/221 = 1/17. Confirmed.
GATE Tip: Without replacement changes the sample space. With replacement, the answer would be 4/52 = 1/13. Always check whether sampling is with or without replacement.

Problem 4: Binomial Distribution

Problem: A multiple-choice test has 20 questions, each with 4 options. A student guesses randomly. Find: (a) Expected number of correct answers. (b) P(exactly 8 correct). (c) Variance.
Approach: X ~ Binomial(n=20, p=1/4). Use formulas: E[X]=np, Var(X)=np(1-p), P(X=k)=C(n,k)pk(1-p)n-k.
Step-by-step Solution:
n = 20, p = 1/4 = 0.25, q = 3/4 = 0.75

(a) E[X] = np = 20 × 0.25 = 5

(b) P(X=8) = C(20,8) × (0.25)8 × (0.75)12
C(20,8) = 125970
(0.25)8 = 1/65536 ≈ 0.00001526
(0.75)12 ≈ 0.031676
P(X=8) = 125970 × 0.00001526 × 0.031676 ≈ 0.0609

(c) Var(X) = npq = 20 × 0.25 × 0.75 = 3.75
Standard deviation = √3.75 ≈ 1.936
GATE Tip: For Binomial: E[X]=np, Var=npq. Mode = ⌊(n+1)p⌋ or (n+1)p-1. Binomial approaches Poisson when n is large and p is small (np = λ).

Problem 5: Poisson Distribution

Problem: On average, a server receives 3 requests per minute. Find: (a) P(exactly 5 requests in a minute), (b) P(at least 2 requests in 30 seconds), (c) Expected time between requests.
Approach: X ~ Poisson(λ). For (a) λ=3, for (b) λ=1.5 (30 seconds). For (c) inter-arrival time is Exponential(1/λ).
Step-by-step Solution:
(a) λ = 3, P(X=5) = e-3 · 35 / 5!
= 0.04979 × 243 / 120 = 0.1008

(b) For 30 seconds: λ = 1.5
P(X ≥ 2) = 1 - P(X=0) - P(X=1)
P(X=0) = e-1.5 = 0.2231
P(X=1) = e-1.5 · 1.5 = 0.3347
P(X ≥ 2) = 1 - 0.2231 - 0.3347 = 0.4422

(c) Inter-arrival ~ Exponential(rate=3)
E[time between requests] = 1/3 minute = 20 seconds
Key Takeaway: Poisson and Exponential are closely related: if arrivals follow Poisson(λ), inter-arrival times follow Exponential(λ). In GATE, remember P(X=0) = e-λ for quick calculations.

Problem 6: Normal Distribution

Problem: Scores on an exam are normally distributed with mean 70 and standard deviation 10. Find: (a) P(score > 85), (b) What score corresponds to the 90th percentile? (c) P(60 < X < 80). Use Z-table values: Φ(1.5)=0.9332, Φ(1.28)=0.8997, Φ(1)=0.8413.
Approach: Standardize: Z = (X - μ) / σ. Use Z-table for probabilities.
Step-by-step Solution:
μ = 70, σ = 10

(a) P(X > 85) = P(Z > (85-70)/10) = P(Z > 1.5)
= 1 - Φ(1.5) = 1 - 0.9332 = 0.0668

(b) 90th percentile: need z such that Φ(z) = 0.90
From table, z ≈ 1.28
X = μ + zσ = 70 + 1.28 × 10 = 82.8

(c) P(60 < X < 80) = P(-1 < Z < 1)
= Φ(1) - Φ(-1) = Φ(1) - (1 - Φ(1)) = 2Φ(1) - 1
= 2(0.8413) - 1 = 0.6826
GATE Tip: The 68-95-99.7 rule: about 68% of data within 1σ, 95% within 2σ, 99.7% within 3σ. Remember Φ(-z) = 1 - Φ(z) by symmetry.

Problem 7: Conditional Expectation

Problem: A box contains 3 red and 2 blue balls. Two balls are drawn without replacement. Let X = number of red balls drawn. Find E[X] using conditional expectation.
Approach: X = X1 + X2 where Xi = 1 if i-th draw is red. Use linearity of expectation. Then verify by computing the full distribution.
Step-by-step Solution:
Method 1: Linearity of Expectation
E[X1] = P(1st is red) = 3/5
E[X2] = P(2nd is red) = 3/5 (by symmetry)
E[X] = E[X1] + E[X2] = 3/5 + 3/5 = 6/5 = 1.2

Method 2: Full Distribution
XOutcomesP(X)
0BB(2/5)(1/4) = 2/20
1RB or BR(3/5)(2/4) + (2/5)(3/4) = 12/20
2RR(3/5)(2/4) = 6/20
E[X] = 0(2/20) + 1(12/20) + 2(6/20) = 24/20 = 6/5 = 1.2. Confirmed.
Key Takeaway: Linearity of expectation holds even for dependent random variables (like sampling without replacement). This is one of the most powerful tools in probability. Always consider indicator random variables for counting problems.

Problem 8: Variance and Covariance

Problem: If X and Y are random variables with E[X]=2, E[Y]=3, Var(X)=4, Var(Y)=9, and Cov(X,Y)=-2, find: (a) E[3X-2Y+5], (b) Var(3X-2Y+5), (c) Correlation coefficient ρ(X,Y).
Approach: Use linearity of expectation, Var(aX+bY) = a2Var(X)+b2Var(Y)+2abCov(X,Y), and ρ = Cov(X,Y)/(σXσY).
Step-by-step Solution:
(a) E[3X - 2Y + 5] = 3E[X] - 2E[Y] + 5 = 3(2) - 2(3) + 5 = 6 - 6 + 5 = 5

(b) Var(3X - 2Y + 5) = 9Var(X) + 4Var(Y) + 2(3)(-2)Cov(X,Y)
= 9(4) + 4(9) + (-12)(-2) = 36 + 36 + 24 = 96

(c) σX = 2, σY = 3
ρ(X,Y) = Cov(X,Y) / (σX · σY) = -2 / (2 × 3) = -1/3 ≈ -0.333
GATE Tip: Constants vanish in variance: Var(X+c) = Var(X). Scaling: Var(aX) = a2Var(X). For independent variables, Cov=0, so Var(X+Y) = Var(X)+Var(Y). Correlation is always in [-1, 1].

Problem 9: Geometric Distribution

Problem: A biased coin has P(Head) = 1/3. It is flipped until the first Head appears. Find: (a) P(first Head on 5th flip), (b) E[number of flips], (c) P(more than 3 flips needed).
Approach: X ~ Geometric(p=1/3). P(X=k) = (1-p)k-1p. E[X] = 1/p.
Step-by-step Solution:
p = 1/3, q = 2/3

(a) P(X=5) = (2/3)4 × (1/3) = 16/81 × 1/3 = 16/243 ≈ 0.0658

(b) E[X] = 1/p = 1/(1/3) = 3
Var(X) = (1-p)/p2 = (2/3)/(1/9) = 6

(c) P(X > 3) = P(first 3 are all Tails) = (2/3)3 = 8/27 ≈ 0.2963
Key Takeaway: The geometric distribution is memoryless: P(X > m+n | X > m) = P(X > n). This makes P(X > k) = qk very easy to compute. In GATE, this property is frequently tested.

Problem 10: Random Variable Transformation

Problem: Let X be uniformly distributed on [0, 2]. Find the PDF and CDF of Y = X2.
Approach: Use the CDF method: FY(y) = P(Y ≤ y) = P(X2 ≤ y), then differentiate for PDF.
Step-by-step Solution:
X ~ Uniform[0,2], so fX(x) = 1/2 for x ∈ [0,2].
Y = X2 ranges from 0 to 4.

CDF of Y:
FY(y) = P(X2 ≤ y) = P(X ≤ √y)  (since X ≥ 0)
= FX(√y) = √y / 2  for 0 ≤ y ≤ 4

PDF of Y:
fY(y) = d/dy [√y / 2] = 1/(2 · 2√y) = 1/(4√y) for 0 < y < 4

Verification: ∫04 1/(4√y) dy = [√y/2]04 = 2/2 - 0 = 1. Valid PDF.
GATE Tip: The CDF method is the safest for transformations. Steps: (1) Find range of Y, (2) Express FY(y) in terms of FX, (3) Differentiate for PDF. For monotone transformations, you can also use: fY(y) = fX(g-1(y)) |dg-1/dy|.

Problem 11: Joint Probability

Problem: Joint PMF of (X,Y) is given. Find marginals, E[X], E[Y], and check independence.

Approach: Sum rows for PX, columns for PY. Check if P(X=x,Y=y) = PX(x)·PY(y) for all x,y.
Step-by-step Solution:
Marginals:
PX(0) = 0.1+0.15+0.05 = 0.30
PX(1) = 0.2+0.25+0.1 = 0.55
PX(2) = 0.05+0.05+0.05 = 0.15

PY(0) = 0.1+0.2+0.05 = 0.35
PY(1) = 0.15+0.25+0.05 = 0.45
PY(2) = 0.05+0.1+0.05 = 0.20

Expected values:
E[X] = 0(0.30)+1(0.55)+2(0.15) = 0.85
E[Y] = 0(0.35)+1(0.45)+2(0.20) = 0.85

Independence check:
PX(0)·PY(0) = 0.30 × 0.35 = 0.105 ≠ P(X=0,Y=0) = 0.1
X and Y are NOT independent.
Key Takeaway: To disprove independence, finding just one pair (x,y) where P(X=x,Y=y) ≠ PX(x)PY(y) is sufficient. To prove independence, you must verify ALL pairs. Uncorrelated does not imply independent (except for jointly normal).

Problem 12: Markov Inequality and Chebyshev

Problem: A random variable X has E[X] = 50 and Var(X) = 25. Using Chebyshev's inequality, find an upper bound on P(|X - 50| ≥ 15). Also find the bound using Markov's inequality if X ≥ 0.
Approach: Chebyshev: P(|X-μ| ≥ kσ) ≤ 1/k2. Markov: P(X ≥ a) ≤ E[X]/a for X ≥ 0.
Step-by-step Solution:
μ = 50, σ2 = 25, σ = 5

Chebyshev:
P(|X - 50| ≥ 15) = P(|X - μ| ≥ 3σ) ≤ 1/32 = 1/9 ≈ 0.1111

Markov:
P(|X - 50| ≥ 15) = P(X ≥ 65 or X ≤ 35)
For the upper tail: P(X ≥ 65) ≤ E[X]/65 = 50/65 = 10/13 ≈ 0.769
This is a weak bound. Markov only gives a useful bound for the upper tail.

Chebyshev gives the tighter bound: P(|X-50| ≥ 15) ≤ 1/9
GATE Tip: Chebyshev is tighter than Markov because it uses variance information. Both give only upper bounds, not exact probabilities. If the distribution is known (e.g., Normal), always compute the exact probability instead.

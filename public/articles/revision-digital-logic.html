2. K-Map Rules

  • Groups must be powers of 2: 1, 2, 4, 8, 16.
  • Make groups as large as possible.
  • Groups can wrap around edges (left-right, top-bottom).
  • Each 1 must be in at least one group. Overlapping is allowed.
  • Don't-cares (X) can be 0 or 1 — use them to make bigger groups.
  • SOP: group the 1s. POS: group the 0s.
  • For n variables: 2-var = 4 cells, 3-var = 8 cells, 4-var = 16 cells.
Remember: Adjacent columns in K-map differ by exactly 1 bit (Gray code): 00, 01, 11, 10.

4. IEEE 754 Floating Point

FormatSignExponentMantissaBiasTotal Bits
Single Precision1 bit8 bits23 bits12732
Double Precision1 bit11 bits52 bits102364
  • Value = (-1)^S * 1.M * 2^(E - bias) (normalized)
  • Denormalized: E = 0, value = (-1)^S * 0.M * 2^(1 - bias)
  • Special: E = all 1s, M = 0 → Infinity. E = all 1s, M ≠ 0 → NaN.
Remember: Bias for single = 127, double = 1023. Actual exponent = stored - bias.

  • 2's complement of N: invert all bits, add 1.
  • Overflow in 2's complement: occurs when adding same-sign numbers gives different-sign result.

6. Combinational Circuits Quick Facts

  • Multiplexer: 2^n inputs, n select lines, 1 output. Can implement any n-variable function.
  • Decoder: n inputs, 2^n outputs. Each output = one minterm.
  • Full Adder: Sum = A XOR B XOR Cin. Carry = AB + BCin + ACin.
  • n-bit Ripple Carry Adder: delay = 2n gate delays (for carry propagation).

7. Sequential Circuits Quick Facts

  • Mod-N counter: needs ceil(log2(N)) flip-flops.
  • Ring counter: N flip-flops for mod-N. Johnson counter: N flip-flops for mod-2N.
  • Mealy machine: output depends on state AND input. Moore machine: output depends only on state.
  • Moore has at least as many states as equivalent Mealy (often more).
Focus Areas: K-map minimization, flip-flop conversions, IEEE 754 representation, and 2's complement overflow detection are the most commonly tested topics.

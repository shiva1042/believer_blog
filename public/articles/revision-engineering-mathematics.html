1. Propositional Logic

  • Tautology: always true. Contradiction: always false. Contingency: neither.
  • p → q ≡ ¬p ∨ q ≡ ¬q → ¬p (contrapositive)
  • De Morgan: ¬(p ∧ q) = ¬p ∨ ¬q; ¬(p ∨ q) = ¬p ∧ ¬q
  • Modus Ponens: p, p→q, therefore q
  • Modus Tollens: ¬q, p→q, therefore ¬p
Remember: p → q is FALSE only when p is TRUE and q is FALSE.

2. Sets, Relations, Functions

  • |P(A)| = 2^|A|. Number of relations on A = 2^(|A|^2).
  • Reflexive: (a,a) for all a. Symmetric: (a,b) implies (b,a). Transitive: (a,b) and (b,c) implies (a,c).
  • Equivalence relation = reflexive + symmetric + transitive. Partitions the set.
  • Partial order = reflexive + antisymmetric + transitive.
  • Total functions from A to B: |B|^|A|. Injections: P(|B|, |A|). Surjections: use inclusion-exclusion.

4. Graph Theory

  • Handshaking: Σdeg(v) = 2|E|
  • Tree: connected + |E| = |V| - 1
  • Euler circuit: connected + all even degrees. Euler path: exactly 2 odd-degree vertices.
  • Planar: E ≤ 3V - 6. Bipartite planar: E ≤ 2V - 4. Euler: V - E + F = 2.
  • Chromatic number: K_n needs n colors. Bipartite (no odd cycle) needs 2.
  • Spanning trees of K_n: n^(n-2)

6. Calculus

  • Rolle's theorem: f(a) = f(b), f continuous on [a,b], differentiable on (a,b) ⇒ exists c where f'(c) = 0.
  • Mean Value Theorem: f'(c) = [f(b)-f(a)]/(b-a).
  • Maxima/minima: f'(x)=0, f''(x)>0 is minimum, f''(x)<0 is maximum.
  • Integration: area under curve. Fundamental theorem: integral of f'(x) = f(b) - f(a).

8. Number Theory

  • GCD via Euclidean: GCD(a,b) = GCD(b, a mod b). GCD * LCM = a * b.
  • φ(n) = n * ∏(1 - 1/p) for each prime factor p of n.
  • Euler's theorem: a^φ(n) ≡ 1 (mod n) when GCD(a,n) = 1.
  • Fermat's little: a^(p-1) ≡ 1 (mod p) for prime p, GCD(a,p) = 1.
  • CRT: system of congruences with coprime moduli has unique solution mod product.

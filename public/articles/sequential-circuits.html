Sequential Circuits

🔲 Section 2: Digital Logic

1. Introduction

Sequential circuits differ from combinational circuits because their output depends on both the current inputs and the current state (memory). They contain feedback paths and storage elements (latches or flip-flops). Sequential circuits can be classified as synchronous (state changes occur on clock edges) or asynchronous (state changes occur whenever inputs change).

The D latch eliminates the invalid state by using a single data input D. When Enable is high, Q follows D. When Enable is low, Q holds its previous value.

3. Flip-Flops

Flip-flops are edge-triggered (positive or negative edge of clock). They change state only at the active clock edge, unlike latches which are level-sensitive.

4. Counters

4.1 Asynchronous (Ripple) Counter

In a ripple counter, each flip-flop is triggered by the output of the previous flip-flop. The clock is applied only to the first flip-flop.

stateDiagram-v2 [*] --> S000 : Reset S000 --> S001 : CLK S001 --> S010 : CLK S010 --> S011 : CLK S011 --> S100 : CLK S100 --> S101 : CLK S101 --> S110 : CLK S110 --> S111 : CLK S111 --> S000 : CLK

Propagation Delay: For an n-bit ripple counter, the maximum delay is n × tpd (flip-flop propagation delay). This limits the maximum operating frequency.

4.2 Synchronous Counter

All flip-flops receive the clock simultaneously. State changes are synchronized, eliminating cumulative delay. Faster but requires more complex logic.

4.3 Up/Down Counter

A control signal determines whether the counter increments or decrements on each clock edge.

4.4 Modulo-N Counter

A mod-N counter counts from 0 to N-1 and then resets. It requires ceil(log2(N)) flip-flops.

5. Registers

5.1 Shift Registers

A shift register stores n bits and shifts them by one position on each clock pulse.

TypeInputOutputApplication
SISO (Serial In, Serial Out)SerialSerialDelay line
SIPO (Serial In, Parallel Out)SerialParallelSerial-to-parallel conversion
PISO (Parallel In, Serial Out)ParallelSerialParallel-to-serial conversion
PIPO (Parallel In, Parallel Out)ParallelParallelTemporary storage

5.2 Universal Shift Register

A universal shift register supports all four modes (SISO, SIPO, PISO, PIPO) and can shift left, shift right, or load data in parallel. Two mode select lines control the operation.

5.3 Ring Counter and Johnson Counter

Ring Counter: A shift register with feedback from the last flip-flop output to the first flip-flop input. An n-bit ring counter has n states (mod-n).

Johnson Counter (Twisted Ring): The complemented output of the last flip-flop feeds back to the first. An n-bit Johnson counter has 2n states (mod-2n).

8. Design Procedure for Synchronous Sequential Circuits

  1. Obtain the state diagram from the problem specification.
  2. Derive the state table.
  3. Perform state minimization (if needed).
  4. Assign binary codes to states (state assignment).
  5. Choose flip-flop type and derive excitation equations using excitation tables.
  6. Draw the circuit.

9. GATE Exam Tips

  • Know all four flip-flop characteristic and excitation tables by heart.
  • For counting problems: n flip-flops can have at most 2n states.
  • Minimum number of flip-flops for mod-N counter: ceil(log2N).
  • Frequency of MSB in a mod-2n counter: fclk / 2n.
  • Race condition can occur in SR latches when both inputs are 1.
  • Edge-triggered flip-flops are immune to glitches except at the clock edge.
  • Master-slave flip-flops sample on one edge and output on the opposite edge.

1. Power Set

Definition: The power set P(A) of a set A is the set of all subsets of A, including the empty set and A itself.

  • If |A| = n, then |P(A)| = 2^n.
  • P({}) = {{}}, so |P({})| = 1.
  • P({a}) = {{}, {a}}, so |P({a})| = 2.
  • P({a, b}) = {{}, {a}, {b}, {a,b}}, so |P({a,b})| = 4.

Example: If A = {1, 2, 3}, then |P(A)| = 2^3 = 8.

P(A) = { {}, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3} }
GATE Favorite: |P(P({}))| = |P({{}})| = 2^1 = 2. And |P(P(P({})))| = 2^2 = 4. This nesting is frequently asked.

2. Cartesian Product

Definition: A x B = {(a, b) | a in A, b in B}

  • |A x B| = |A| * |B|
  • A x B is NOT equal to B x A (unless A = B or one is empty).
  • A x {} = {} (Cartesian product with empty set is empty).
  • A x (B ∪ C) = (A x B) ∪ (A x C) — distributes over union.

Example: A = {1, 2}, B = {a, b, c}.

A x B = {(1,a), (1,b), (1,c), (2,a), (2,b), (2,c)}
|A x B| = 2 * 3 = 6

3. Countable vs Uncountable Sets

3.1 Definitions

TypeDefinitionExamples
FiniteHas a fixed number of elements{1, 2, 3}, {a, b}
Countably InfiniteCan be put in 1-1 correspondence with natural numbersN, Z, Q, set of all strings over finite alphabet
UncountableCannot be enumerated by natural numbersR, [0,1], P(N), set of all real functions

3.2 Key Results

  • N (natural numbers) is countably infinite. Cardinality = aleph-naught.
  • Z (integers) is countably infinite (enumerate: 0, 1, -1, 2, -2, 3, -3, ...).
  • Q (rationals) is countably infinite (Cantor's diagonal argument / zig-zag enumeration).
  • R (reals) is uncountable (proved by Cantor's diagonal argument).
  • Union of countably many countable sets is countable.
  • Cartesian product of two countable sets is countable.
  • Power set of a countably infinite set is uncountable: |P(N)| = 2^(aleph-naught) = cardinality of R.

4. Cantor's Diagonal Argument

This proves that the set of real numbers in [0, 1] is uncountable.

Proof by contradiction:

  1. Assume [0,1] is countable. Then we can list all reals: r1, r2, r3, ...
  2. Write each real in decimal expansion: 
    r1 = 0.d(1,1) d(1,2) d(1,3) ...
r2 = 0.d(2,1) d(2,2) d(2,3) ...
r3 = 0.d(3,1) d(3,2) d(3,3) ...
...
  3. Construct a new number s = 0.s1 s2 s3 ... where s_i differs from d(i,i) (e.g., s_i = d(i,i) + 1 mod 10, avoiding 0 and 9 to prevent issues with dual representations).
  4. Then s differs from every r_i in at least the i-th digit.
  5. So s is in [0,1] but not in our list. Contradiction.

Therefore [0,1] (and hence R) is uncountable.

5. Cardinality of Infinite Sets

SymbolNameSets with this Cardinality
aleph-naughtCountably infiniteN, Z, Q, set of all finite strings, set of all programs
c (= 2^(aleph-naught))Cardinality of continuumR, [0,1], P(N), set of all functions from N to {0,1}
  • Cantor's theorem: For any set A, |P(A)| > |A|. There is no surjection from A to P(A).
  • This means there is an infinite hierarchy of infinities: |N| < |P(N)| < |P(P(N))| < ...
GATE Application: The set of all languages over a finite alphabet is uncountable, but the set of all Turing machines is countable. Therefore, there exist languages that are not recognizable by any Turing machine.

6. Inclusion-Exclusion Principle

6.1 Formula for Two Sets

|A ∪ B| = |A| + |B| - |A ∩ B|

6.2 Formula for Three Sets

|A ∪ B ∪ C| = |A| + |B| + |C| - |A∩B| - |B∩C| - |A∩C| + |A∩B∩C|

6.3 General Formula

|A1 ∪ A2 ∪ ... ∪ An| = Σ|Ai| - Σ|Ai ∩ Aj| + Σ|Ai ∩ Aj ∩ Ak| - ... + (-1)^(n+1)|A1 ∩ ... ∩ An|

6.4 Example

Q: In a class of 100 students, 40 study Math, 30 study Physics, 20 study Chemistry. 10 study both Math and Physics, 8 study both Physics and Chemistry, 5 study both Math and Chemistry, 3 study all three. How many study at least one subject?

|M ∪ P ∪ C| = 40 + 30 + 20 - 10 - 8 - 5 + 3 = 70

70 students study at least one subject.
30 students study none.

Q: How many integers from 1 to 1000 are divisible by 2, 3, or 5?

|A2| = floor(1000/2) = 500
|A3| = floor(1000/3) = 333
|A5| = floor(1000/5) = 200
|A2 ∩ A3| = floor(1000/6) = 166
|A2 ∩ A5| = floor(1000/10) = 100
|A3 ∩ A5| = floor(1000/15) = 66
|A2 ∩ A3 ∩ A5| = floor(1000/30) = 33

Answer = 500 + 333 + 200 - 166 - 100 - 66 + 33 = 734

7. Pigeonhole Principle

7.1 Basic Principle

If n items are placed into m containers and n > m, at least one container has more than one item.

Generalized: If n items in m containers, at least one container has at least ceil(n/m) items.

7.2 Advanced Applications

Example 1: Among any 13 people, at least 2 were born in the same month.

13 people, 12 months. By pigeonhole, at least ceil(13/12) = 2 share a month.

Example 2: In any group of 6 people, either 3 mutually know each other or 3 mutually do not know each other.

This is Ramsey theory: R(3,3) = 6. Consider one person P. By pigeonhole, P either knows 3+ people or does not know 3+ people. In either case, among those 3, check pairwise relationships.

Example 3: Pick 5 numbers from {1, 2, ..., 8}. Prove two of them sum to 9.

Pairs that sum to 9: {1,8}, {2,7}, {3,6}, {4,5}.
That's 4 "pigeonholes". Picking 5 numbers from 8,
by pigeonhole, two must come from the same pair.
Those two sum to 9.

8. De Morgan's Laws for Sets

LawStatement
De Morgan's 1(A ∪ B)' = A' ∩ B'
De Morgan's 2(A ∩ B)' = A' ∪ B'
Generalized 1(∪ Ai)' = ∩ Ai'
Generalized 2(∩ Ai)' = ∪ Ai'

In words: The complement of a union is the intersection of complements, and vice versa.

Proof sketch (De Morgan's 1): x is in (A∪B)' iff x is not in A∪B iff x is not in A AND x is not in B iff x is in A' AND x is in B' iff x is in A'∩B'.

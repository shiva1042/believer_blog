3. Example 1: Simple Loop

for (int i = 0; i < n; i++) {
    // O(1) work
}

Analysis: i goes 0, 1, 2, ..., n-1
  Loop runs n times.
  Answer: O(n)

4. Example 2: Logarithmic Loop

for (int i = 1; i < n; i *= 2) {
    // O(1) work
}

Analysis: i takes values 1, 2, 4, 8, ..., until i >= n
  Number of iterations: log2(n)
  Answer: O(log n)

5. Example 3: Nested Independent Loops

for (int i = 0; i < n; i++) {          // n times
    for (int j = 0; j < m; j++) {      // m times each
        // O(1) work
    }
}

Analysis: Outer runs n times, inner runs m times per outer iteration.
  Total: n * m
  Answer: O(n * m), or O(n^2) if m = n

6. Example 4: Dependent Nested Loops

for (int i = 0; i < n; i++) {
    for (int j = 0; j < i; j++) {
        // O(1) work
    }
}

Analysis: Inner loop runs 0 + 1 + 2 + ... + (n-1) times
  Sum = n*(n-1)/2
  Answer: O(n^2)

7. Example 5: Log Inside Linear

for (int i = 0; i < n; i++) {          // n times
    for (int j = 1; j < n; j *= 2) {   // log(n) times
        // O(1) work
    }
}

Analysis: n * log(n)
  Answer: O(n log n)

8. Example 6: Tricky Dependent Loop

for (int i = 1; i < n; i *= 2) {
    for (int j = 0; j < i; j++) {
        // O(1) work
    }
}

Analysis: Inner loop runs i times.
  i takes values: 1, 2, 4, 8, ..., n
  Total = 1 + 2 + 4 + 8 + ... + n = 2n - 1

  Answer: O(n)  (NOT O(n log n)!)
Common Mistake: Students often see a log loop with a linear loop and assume O(n log n). When the inner loop depends on the outer variable, you must SUM the series, not multiply.

9. Example 7: Nested Log Loops

for (int i = 1; i < n; i *= 2) {       // log(n)
    for (int j = 1; j < n; j *= 3) {   // log3(n)
        // O(1) work
    }
}

Analysis: log2(n) * log3(n) = O(log^2(n))
  Answer: O(log^2 n)

10. Example 8: Square Root Loop

for (int i = 1; i * i < n; i++) {
    // O(1) work
}

Analysis: Loop runs while i^2 < n, i.e., i < sqrt(n)
  Answer: O(sqrt(n))

11. Example 9: Amortized Analysis — Dynamic Array

// Appending n elements to a dynamic array that doubles when full
for (int i = 0; i < n; i++) {
    arr.append(i);  // occasionally triggers resize
}

Analysis:
  Resizes at sizes 1, 2, 4, 8, ..., n
  Copy costs: 1 + 2 + 4 + 8 + ... + n = 2n - 1
  Regular appends: n
  Total: 3n - 1

  Answer: O(n) total, O(1) amortized per append

12. Example 10: Recursive — Simple

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

Recurrence: T(n) = T(n-1) + O(1)
  T(n) = T(n-1) + c = T(n-2) + 2c = ... = nc
  Answer: O(n)

13. Example 11: Recursive — Binary Recursion

int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}

Recurrence: T(n) = T(n-1) + T(n-2) + O(1)
  T(n) >= 2*T(n-2) => T(n) = O(2^(n/2))
  Tighter bound: T(n) = O(phi^n) where phi = (1+sqrt(5))/2 ≈ 1.618
  Simplified: O(2^n)

14. Example 12: Master Theorem Application

Master Theorem: T(n) = a*T(n/b) + O(n^d)

Case 1: If d < log_b(a): T(n) = O(n^(log_b(a)))
Case 2: If d = log_b(a): T(n) = O(n^d * log n)
Case 3: If d > log_b(a): T(n) = O(n^d)

Example: Merge Sort
  T(n) = 2*T(n/2) + O(n)
  a=2, b=2, d=1
  log_2(2) = 1 = d => Case 2
  T(n) = O(n log n)

Example: Binary Search
  T(n) = T(n/2) + O(1)
  a=1, b=2, d=0
  log_2(1) = 0 = d => Case 2
  T(n) = O(log n)

Example: Strassen's Matrix Multiply
  T(n) = 7*T(n/2) + O(n^2)
  a=7, b=2, d=2
  log_2(7) ≈ 2.807 > 2 => Case 1
  T(n) = O(n^(log_2(7))) ≈ O(n^2.807)

15. Example 13: Loop with Multiple Variables

int i = 0, j = 0;
while (i < n) {
    if (j < n) {
        j++;
    } else {
        i++;
        j = 0;
    }
}

Analysis: Each iteration either increments j or resets j and increments i.
  i goes from 0 to n: incremented n times.
  For each value of i, j goes from 0 to n: n increments.
  Total iterations: n * n + n = O(n^2)

16. Example 14: While Loop with Division

int i = n;
while (i > 1) {
    i = i / 2;
}

Analysis: i takes values n, n/2, n/4, ..., 1
  Iterations: log2(n)
  Answer: O(log n)

17. Example 15: Complex Nested with Harmonic Series

for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= n; j += i) {
        // O(1) work
    }
}

Analysis: For each i, inner loop runs n/i times.
  Total = n/1 + n/2 + n/3 + ... + n/n
        = n * (1 + 1/2 + 1/3 + ... + 1/n)
        = n * H(n)  where H(n) is the harmonic series
        = n * O(log n)

  Answer: O(n log n)
GATE Tip: The harmonic series 1 + 1/2 + 1/3 + ... + 1/n = O(log n). This pattern appears in sieve algorithms and many GATE problems. Recognize the pattern: inner loop steps by outer loop variable.
